# Art Gallery

Starter React app (Vite) for the Art Gallery project. This repo now includes a minimal React scaffold with a `Navbar` and `Footer`.

Files added
- `package.json` - project metadata and scripts
- `index.html` - app entry HTML
- `src/main.jsx` - React entry
- `src/App.jsx` - main app layout (uses `Navbar` and `Footer`)
- `src/components/Navbar.jsx` - top navigation
- `src/components/Footer.jsx` - footer
- `src/index.css` - basic styles

Quick start

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

Open the URL shown by Vite (usually `http://localhost:5173`).
